**CAUClub**
============
## **목적**
#### 동아리 활동 인증서를 발급함에 있어 동연 집행부가 직접 엑셀 파일을 확인하고 수기로 인증서를 발급하던 기존 방식을, 웹 및 DB를 통해 자동화하여 수요자와 동연 모두 편하게 업무를 처리할 수 있도록 함
#### (본격 맨 땅에 헤딩하기 프로젝트...)
------------
## **사용 스택**

* Back-end: **Django**
